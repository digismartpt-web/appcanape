# App Canapé

## Description
 Application de réservation et gestion pour App Canapé.

## Stack Technique
- React
- Vite
- TypeScript
- Tailwind CSS
- Supabase
- Zustand
- Google Maps
- React Hook Form
- Zod

## Commandes
- Installer les dépendances : `npm install`
- Démarrer le serveur de développement : `npm run dev`
- Générer la build : `npm run build`

## Configuration
- Ajoutez un fichier `.env.local` avec les variables suivantes :
  - `VITE_SUPABASE_URL=https://ton-url-supabase-vps.com`
  - `VITE_SUPABASE_ANON_KEY=ta-cle-publique`
  - `VITE_GOOGLE_MAPS_API_KEY=ta-cle-api-google`
